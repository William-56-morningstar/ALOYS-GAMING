# 🧠 ALOYS-GAMING (OmniQuiz Hub)

Une plateforme web interactive regroupant des modules de quiz dynamiques et des défis intellectuels.

## 📂 Organisation du Projet
Le projet est structuré de manière modulaire pour une maintenance facile :
* **math-storm** : Module de calcul mental et défis mathématiques.
* **quiz-culture** : Module de culture générale et capitales.
* **js/core.js** : Le moteur principal qui gère la logique globale.

## 🌟 Points Forts
* **Génération Dynamique** : Algorithmes JavaScript pour des questions illimitées.
* **Interface Mobile** : Design optimisé pour les écrans tactiles.
* **Architecture Modulaire** : Facilité d'ajout de nouveaux thèmes de quiz.

## 🛠️ Stack Technique
* HTML5 / CSS3 (Design moderne)
* JavaScript ES6+ (Logique métier)
* GitHub Pages (Hébergement)
